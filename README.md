# Test task 

## Getting Started 

1) Install packages 
```
npm install
```

2) Setup environment variables 
- run 
```
cp .env_local .env
```

- set the values for each env variable, in our case the random facts url is: https://uselessfacts.jsph.pl/random.json 

3) Run the project in development mode 
```
npm start 
```
